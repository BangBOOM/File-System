# 操作系统课程设计

多用户、多级目录结构文件系统的设计与实现。

模拟实现UNIX的文件系统



## git使用规范

提交文件的时候几个规范的写法：

```
init: 初始化项目
try: 尝试完成xxx功能 
add: 添加用xxx功能
update: 完善xx功能
done: 完成xx功能
fix: 修复xxbug
delete: 删除xxx内容
reset: 放弃修改返回
```
